Grundlæggende web

Tema 2 omhandler det grundlæggende ved webudvikling og webdesign. I dette tema skulle jeg kode min første hjemmeside bestående af 5 html sider ved brug af css styling, wireframes, layoutdiagrammer samt layoutmodeller: grid og flexbox

Process

I dette tema lærte vi om wireframes og layout diagrammer, og hvordan de aflæses med henblik på kodning af hjemmesiden. Dernæst blev vi introduceret til layout modellerne grid og flex, der anvendes i layoutet af og opdelingen af elementer på hjemmesiden.

I dette tema har jeg - ved afkodningen af hjemmesidens layout diagram - anvendt både grid og flex til layout og design af hjemmesiden. Jeg har gjort brug af grid til opdeling af hjemmesiden i form af rækker og kolonner, og har anvendt flex til at placere billeder og tekst mere fleksibelt på siden. Resultatet af grid ses på hjemmesiden på siden ‘Operativsystemer’, mens resultatet af flex box ses på siden ‘Komponenter’.

Viden

Fra dette tema har jeg taget brugbar viden og nye kompetencer med mig. En viden og egenskaber, der er essentielle at have med sig til kommende projekter. Jeg har lært, hvordan man aflæser et wireframe samt layout diagrammer, og koder ud fra disse. Derudover har jeg lært, hvordan man gør brug af css til styling af hjemmesiden, samt grid og flex til layoutet og opdeling af elementer på hjemmesiden.
